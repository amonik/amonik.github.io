Installing KVM

On CentOS 7.4

Procedure:
yum install epel-release
yum update
yum groupinstall "Development Tools"
yum groupinstall "Virtualization Tools"
yum install virt-install libvirt-python virt-manager virt-install libvirt-client

Build 3 VMs for Eleasticsearch, Logstash, and Kibana (ELK)

Install Juniper vMX on KVM

Push routing configuration to Juniper vMX devices using API and Python

Install Linux Containers or Kunbernetes/Docker

Setup a syslog server on a VM

Configure vMX, Containers, and VMs to send logs to syslog server

Setup monitoring of vMX, VMs, and containers using ELK

Rebuild the setup using Ansible/Python
